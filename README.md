- Student Login
- Teacher Login
- Admin Panel
- Student Registration
- Attendance Management
- Leave Management
- Holiday Calendar
- Notice Board
- Attendance Reports
- Dashboard with Analytics
- Export to PDF & Excel
- Responsive UI
- Dark Mode
